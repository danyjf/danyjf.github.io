Space Warfare (Unreal Engine C++)

This is a multiplayer project developed by myself using Unreal Engine. In this game the player can place satellites in orbit and control them to try to intercept other satellites or asteroids. All movement in the game is based on an implementation of realistic newtonian physics in space.

There are two available game modes. The Casual and Bots. In the Casual game mode the player manually controls each individual satellite. In the Bots game mode, the game allows the player to access and send information to the game through HTTP requests, allowing the player to program their own bot to play the game.

Key Features Implemented

  • A multiplayer solution that allows up to four players to play together, where one player acts as the host of a session
  • Gravity simulation by applying Newton's law of universal gravitation to all bodies in the simulation
  • Orbit visualization system that based on the current position and velocity of a body, displays the orbit that will be followed by it
  • Asteroid spawn system, that spawns an asteroid on a random earth orbit every couple seconds
  • Restful API, that allows the player to access and send information to the game through an API, allowing players to implement their own algorithms to control satellites

Virulent Odissey (Unity C#)

This is a VR project developed primarily for the headset Meta Quest 1 and 2, by a team of two elements using the Unity engine. In this prototype, the player plays from the perspective of a virus and the goal is to reach a cell and complete a replication cycle of the virus.

The player starts in an area outside the cell where the player will avoid different defence organisms and learn about them. Then, upon reaching the cell, the player will infect it by getting inside of the cell. When inside the cell, the player will learn about and perform the steps necessary for a full replication cycle.

Main Responsabilities

  • Implement movement system, using grab and drag movement when the player is outside the cell, and joystick movement when inside the cell
  • Implement narrator system that guides the player on what to do
  • Implement section where player has to dodge enemies that are spawned randomly moving in the direction of the player
  • Implement player interaction with the cell, where player needs to connect with the cell with both hands

Avoid (Unity C#)

This is a mobile game project developed by myself using the Unity engine. In this game the player controls a spaceship using joystick controls on the phone UI. The weapons on the space ship fire automatically in the direction of the closest enemy, allowing the player to focus most of the atention on avoiding the enemy projectiles. Different enemies fire different projectiles, for example, there are homing missiles that follow the player until they explode.

The game also has an upgrade system that allows the player to increase stats like the damage of the weapon or the amount of health of the spaceship. To purchase this upgrades the player must collect resources inside the levels, harder levels provide more resources but have a higher risk, since when the player doesn't destroy all enemies in a level, the player does not obtain the collected resources.

Key Features Implemented

  • Space ship controls
  • Save system, saving the amount of resources the player has, which levels have been unlocked and the upgrade levels of the player
  • Different types of weapon, for example, the player weapons which aims automatically at the enemies, the homing missile weapon and the lazer weapon
  • Upgrade system allowing the player to purchase upgrades for the shield, the gun, the resource storage, and the length of the collector trail
  • Resource collection system consisting of a trail behind the player that detects when a resource has collided with it and collects the resources

Break The Game (Unity C#)

Break the Game is a 2D puzzle game developed by myself in two days using the Unity game engine. In the game, the player has to find a "bug" in each level in order to reach the goal without killing the enemy.
